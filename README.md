# Paxful Application 

 This repository contains the paxful app using python and flask. 

## To run and install the dependencies of the REST app:


```bash
$ bash run.sh 
```

## Before you begin

### Prerequisites

- Kubernetes 1.19+
- Helm 3.2.0+
- minikube
- SMTP server
